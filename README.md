# 🎮 Custom Web Games 🕹️

A collection of lightweight, responsive, and interactive web games built entirely using vanilla web technologies. Perfect for quick browser gaming without any installations! 🚀

Here is the complete list of games available in this repository:

| **SpeedyVerse** ⚡ | Fast-paced gameplay / arcade runner. | HTML, CSS, JS |
