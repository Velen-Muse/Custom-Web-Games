# 🚀 **Getting Started**

Running any game locally takes just a few steps.

## **1. Clone the Repository**

Open your terminal and run:

```bash
git clone https://github.com/Velen-Muse/Custom-Web-Games.git
cd Custom-Web-Games
```

## **2. Choose a Game**

Navigate to the folder of the game you'd like to play.

## **3. Launch the Game**

Open the **`index.html`** file in your browser, or launch it with **Live Server** for the best development experience.
